# Mealie Recipe Exchange

Exports recipes to JSON files. Recipes are exported when a recipe does not exist or when the recipe has been modified. The script requires three inputs:

1. The URL for Mealie (Environment Variable: `MEALIE_URL`)
1. The API Token for Mealie (Environment Variable: `MEALIE_API_TOKEN`)
1. The Export directory for the recipes (Environment Variable: `MEALIE_EXPORT_DIR`)
